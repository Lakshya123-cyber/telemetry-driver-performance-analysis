# Telemetry Driver Performance Analysis

Python application developed as part of my Bachelor's Thesis at **Satakunta University of Applied Sciences (SAMK)**.

The application analyses telemetry data exported from **Assetto Corsa** to evaluate driver performance through lap comparison, sector analysis, racing line reconstruction, and automated driver feedback.

---

## Features

- Automatic telemetry preprocessing
- Lap detection and validation
- Exact sector timing using interpolation
- Lap and sector performance analysis
- Racing line deviation analysis
- Brake, throttle, steering, and G-force analysis
- Automated driver feedback generation
- Interactive visualisations and CSV reports
